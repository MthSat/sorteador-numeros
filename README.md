# Sorteador de Números

Este é um projeto criado no curso **Lógica de Programação: Praticando Desafios** da Alura. O objetivo é implementar a funcionalidade de um sorteador de números utilizando JavaScript, em uma página que já possui HTML e CSS prontos.

## Funcionalidades

- O usuário define:
  - Quantos números deseja sortear.
  - O valor inicial e o valor final do intervalo de sorteio.
- O JavaScript então sorteia os números dentro das configurações informadas e os exibe na tela.

## Tecnologias Utilizadas

- **HTML**: Estrutura básica do site.
- **CSS**: Estilos para uma apresentação visual agradável.
- **JavaScript**: Lógica do sorteio de números.

## Como Usar

1. Abra o projeto em um navegador.
2. Informe o intervalo (valor inicial e valor final) e quantos números deseja sortear.
3. Clique no botão de sorteio para gerar os números aleatórios.
